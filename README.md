# 🎓 考试倒计时

一个精美的Web考试倒计时工具，帮助你管理考试时间，保持高效学习！

## ✨ 功能特性

- ⏱️ **实时倒计时** - 精确到秒的倒计时显示
- 🎨 **精美界面** - 动态渐变背景、粒子动画效果
- 📱 **响应式设计** - 完美适配各种设备
- 💾 **数据持久化** - 考试数据自动保存到本地
- 🔄 **多考试管理** - 支持添加多个考试
- 💫 **动态效果** - 数字变化动画、闪烁分隔符

## 🛠️ 本地运行

```bash
# 克隆仓库
git clone https://github.com/LiuQin123616/exam.git
cd exam

# 启动本地服务器
python -m http.server 8000

# 打开浏览器访问
# http://localhost:8000
```

## 📖 使用方法

1. **添加考试** - 点击右下角的 `+` 按钮
2. **填写信息** - 输入考试名称、日期和时间
3. **确认添加** - 点击"确认添加"按钮
4. **查看倒计时** - 选择考试查看实时倒计时
